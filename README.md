# Question Explorer

A client-side Next.js application for exploring interview questions with
filtering, sorting, and detailed views.

## Live Demo

https://question-explorer.vercel.app

## Tech Stack

- Next.js (App Router)
- TypeScript
- Material UI
- PapaParse

## Features

- Search, filter, and sort questions
- Individual question detail pages
- Responsive two-column layout
- Client-side CSV data loading
